# Product Context

## Why this project exists
Impact Framework（IF）は、Green Software Foundationによって開発されたプロジェクトで、ソフトウェアの環境影響を測定・分析するためのツールです。このプロジェクトは、ソフトウェア開発者が自分たちのアプリケーションの環境影響を理解し、より持続可能なソフトウェアを構築するのを支援することを目的としています。

## What problems it solves
- ソフトウェアの環境影響を標準化された方法で測定する
- 異なるアプリケーションやシステム間で環境影響を比較できるようにする
- 開発者が環境に配慮したデザイン決定を行うためのデータを提供する
- 環境影響を継続的にモニタリングし、改善するためのフレームワークを提供する

## How it should work
Impact Frameworkは、YAMLまたはJSONで記述されたマニフェストを処理し、環境影響の計算を行います。Web APIサーバー（if-api）を通じて、ユーザーはHTTPリクエストを送信してマニフェストを処理し、結果を取得できます。

APIは以下の形式のリクエストを受け付けます：
- JSON形式
- YAML形式
- multipart/form-data形式

処理されたマニフェストは、環境影響の計算結果を含むJSONレスポンスとして返されます。
